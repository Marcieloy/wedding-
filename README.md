# Rosa & John's Wedding Website

A beautiful, elegant wedding invitation website for Rosa and John's wedding on July 20, 2026.

## Features

- 📧 **RSVP Form** - Guests can easily respond to the invitation
- 📍 **Reception Details** - Complete venue information and map
- 💍 **Elegant Design** - Responsive and mobile-friendly
- ✨ **Beautiful Typography** - Professional calligraphy and serif fonts

## Pages

1. **Invitation** (`index.html`) - Main invitation with wedding entourage
2. **RSVP** (`rsvp.html`) - Response form for guests
3. **Reception** (`reception.html`) - Venue and celebration details

## RSVP Storage

The RSVP form data is currently saved to the browser's localStorage. For production, you may want to integrate with:
- Formspree (https://formspree.io/)
- Google Forms
- A custom backend service
